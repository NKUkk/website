Leetcode — Binary Tree Preorder Traversal
发表于 2020-05-17| 更新于 2020-05-19|编程题
阅读量:|评论数:

Question

Given a binary tree, return the preorder traversal of its nodes’ values.

Example

Input: [1,null,2,3]
1
 \
  2
 /
3
Output: [1,2,3]

Summary

There are two general strategies for traversing the tree:

  • Depth First Search(DFS)
    In this strategy, we take depth as the priority, so that we can reach a certain leaf from the heel, and then return the root to another branch. According to the relative order of root node, left child and right child, the depth first search strategy can be subdivided into pre order traversal, middle order traversal and post order traversal.
  • Breadth First Search(BFS)
    We visit the whole tree layer by layer according to the height order. The high-level nodes will be accessed first than the low-level nodes.
